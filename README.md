# TripleTen Art Gallery (Homeland)

Página dedicada a contar a história de desenvolvedores da TripleTen que vivem ao redor do mundo.

Desenvolvido como projeto por Pollyana Gonçalves para o Bootcamp de Fullstack da TripleTen.

Como melhoria, acredito que seria interessante ter uma seção com o nome dos profissionais para que as pessoas possam acessar de forma organizada sua história, podendo filtrar por pessoa ou por região/país.
